	Neste exemplo aplico algumas técnicas de criação de paletas de cores conhecidas que são 	obtidas através do circulo cromático, algumas delas são:
</p>

	<li>Monocromatismo</li>
	<li>Cores Complementares</li>
	<li>Quadrado</li>
